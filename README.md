### 💻 Featured Project

#### 🔨 [SkillHire](https://github.com/Armaghan-04/skillHire) — Hybrid Freelance Marketplace & Job Management System (FYP)
A robust, dual-purpose ecosystem serving as a comprehensive freelance marketplace and professional job management platform. Built from the ground up with strict software engineering discipline.
*(Note: Codebase is temporarily private during the final evaluation phase and will be made public immediately following my defense).*

*   **Advanced Backend Architecture:** Built using a decoupled **Clean Architecture** approach. Implemented rigid **Role-Based Access Control (RBAC)** across distinct identity levels (Users, Employers, Service Providers, and Admins).
*   **Security & Session Hardening:** Developed a secure authentication pipeline utilizing **JSON Web Tokens (JWT)** combined with automated **Refresh Token Rotation** and token versioning for reliable session management.
*   **High-Performance Data Handling:** Designed and integrated optimized server-side pagination workflows on key resource endpoints to guarantee lightning-fast response times and minimal server overhead.
*   **Dynamic UI/UX:** Built a responsive frontend with **React**, **TypeScript**, and **Redux Toolkit** for centralized state management, styled with a high-readability layout using **Tailwind CSS**.
