### 🔨 Featured Project

**[SkillHire](https://github.com/Armaghan-04/skillHire)** — Final Year Project
> A hybrid freelance marketplace combining professional networking with gig-based hiring. Employers post jobs and services, providers apply and get hired, all through a role-based platform with a unified dashboard API.

- **Backend:** .NET 8 Web API · Clean layered architecture (Repository / Unit of Work / Unit of Service) · EF Core · role-based access across a four-tier hierarchy (User, Employer, ServiceProvider, Admin)
- **Frontend:** React 18 + TypeScript + Redux Toolkit + Tailwind v4, feature-based folder structure
- **Status:** Actively in development with daily commits — currently private during FYP evaluation, will be made public after submission/defense on **[date]**
